# lab-webgl-home

Article about this repository can be found here: https://jolicode.com/blog/webgl-the-wow-effect

## To start :

1. Clone the project & ```cd path/to/the/clonned/repository```
2. Run ```npm install```
3. Run ```npm start```
4. Go to ```localhost:3000``` on your browser.


## To build :

Run ```npm run build```
